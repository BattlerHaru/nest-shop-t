# ShopDB API

1. Clone project

2. install node_modules

```bash
yarn install
```

3. Clone file **.env.template** and rename to **.env** with your credentials

4. Up Database

```bash
docker-compose up -d
```

5. seed execute

```
http://localhost:3000/api/seed
```

6. Run project

```bash
yarn start:dev
```
